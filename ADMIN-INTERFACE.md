## Question
- Question type should be represented by an icon (Lucide)
  - Standard: `circle-question-mark`
  - Boolean: `circle-power`
  - 1–10: `timeline`
  - Choose between: `blocks`
  - Colors: `paint-bucket`
  - Numbers: `binary`
  - Century / Decade: `calendar-days`
